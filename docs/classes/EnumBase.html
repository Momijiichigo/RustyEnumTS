Class EnumBase<M>

Base class for enums that each variants can hold a value.

Generic type M should be:

{
  VariantName: VariantType
}

For example, Result<T, E> class is defined as:

class Result<T, E> extends EnumBase<{ Ok: T, Err: E }>
